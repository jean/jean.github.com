Browser wikis

Wikis that live in a browser.

There's TiddlyWiki, mature and well-thought-through. There are web-hosted options, but you can use it offline entirely, or share privately via Dropbox.

There's the Smallest Federated Wiki by Ward Cunningham, web-only and focussed on federation. A wiki may span servers. It also features simple data visualization and automation (I wonder how this compares to tiddlers). Ward demonstrates running the wiki on Arduinos to publish sensor data for both human and machine consumption.

Now Dave Winer's Little Outliner has shown up. It looks like it shares many properties with these wikis. Storage is local, interchange is via OPML export/import.
